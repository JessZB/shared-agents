# Shared Agents Repository

Centralized repository for Antigravity AI agents (rules, skills, and workflows).

## Structure
- `littlepigman-portafolio/`: Specific rules for the Astro Frontend.
- `yezzfolio/`: Specific rules for the Bridge CMS (Express/React).
- `common/`: Shared skills and global behaviors.

## Usage
Add this repository as a submodule in your project and symlink the `.agents` folder to the corresponding subdirectory.
